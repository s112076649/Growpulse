# GrowPulse用户指南

## 1. 快速开始

### 本地运行

1. 确保已安装所需的依赖：
   ```bash
   # 安装依赖
   yarn install
   ```

2. 运行应用：
   ```bash
   # 一键启动前后端
   ./quick-start.sh
   ```

3. 访问应用：
   - 前端：http://localhost:3002
   - 后端：http://localhost:3100

### 首次登录

1. 访问GrowPulse首页
2. 创建新账户或使用默认管理员账户 (admin@example.com / growthbook)
3. 创建您的第一个组织

## 2. 核心功能使用

### 创建特性标志

1. 导航到"Features"页面
2. 点击"+ New Feature"按钮
3. 填写特性信息：
   - 名称和描述
   - 类型（布尔值、字符串、数字等）
   - 目标规则和条件
4. 保存并获取SDK密钥

### 设置A/B测试实验

1. 导航到"Experiments"页面
2. 点击"+ New Experiment"按钮
3. 配置实验：
   - 名称和目标
   - 变体设置
   - 分流规则
   - 指标选择
4. 启动实验并追踪结果

### 使用高级分析功能

1. **回归调整**：在实验结果页面启用回归调整，提高结果准确性
2. **人群分析**：在指标设置中定义人群分组，查看不同群体的实验效果
3. **多指标分析**：同时分析多个相关指标，获取全面洞察

## 3. 管理团队与项目

### 团队管理

1. 导航到"Settings > Team Management"
2. 邀请团队成员：
   - 输入邮箱地址
   - 设置角色和权限
3. 创建团队分组和权限设置

### 项目分组

1. 导航到"Projects"页面
2. 点击"+ New Project"创建项目
3. 将实验和特性标志分组到不同项目
4. 设置项目级别的权限

## 4. 集成与连接

### 数据源连接

1. 导航到"Settings > Data Sources"
2. 添加数据库连接（支持多种数据库和数据仓库）
3. 配置SQL查询模板和指标定义

### SDK集成

1. 选择适合您技术栈的SDK：
   - JavaScript/React
   - Python
   - Ruby
   - Go
   - PHP
2. 按照集成指南将SDK添加到您的应用
3. 验证SDK连接状态

## 5. 常见问题解决

### 实验未显示数据

- 检查数据连接配置
- 验证事件跟踪代码
- 确认实验分流设置

### 特性标志不生效

- 检查SDK配置
- 验证特性键名是否正确
- 确认目标规则设置正确

### 性能优化

- 使用缓存减少数据库查询
- 优化大型实验的数据加载
- 设置合理的指标查询窗口
